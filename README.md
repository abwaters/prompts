# prompts

A collection of prompts I've created that I think are interesting.

> ## Contributions welcome
>
> These are living experiments — I fully expect them to change. If you spot a mistake, have a better idea, or want to suggest new behavior, please **[open a pull request](https://github.com/abwaters/prompts/pulls)** or **[file an issue](https://github.com/abwaters/prompts/issues)** with your recommendation. Changes to a prompt, its README, or its design are all fair game.

## Structure

Each prompt lives in its own folder containing:

- `README.md` — what it does, how it behaves, and a link to the live GPT.
- `prompt.md` — the prompt text itself.
- `design-notes.md` — why it's written the way it is: goals, tradeoffs, and the tensions accepted on purpose.

Plus any supporting files a prompt needs.
