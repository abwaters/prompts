# prompts

A collection of prompts I've created that I think are interesting.

## Contents

- [`chatgpt-gpts/`](chatgpt-gpts/) — prompts for ChatGPT custom GPTs.

## Structure

Each prompt is its own folder containing:

- `README.md` — what the prompt does, how to use it, and any setup (e.g. GPT configuration).
- `prompt.md` — the prompt text itself.
- `design-notes.md` — why it's written the way it is: goals, iterations, tradeoffs, and things that didn't work.

Plus any supporting files a prompt needs (example conversations, knowledge files, etc.).
